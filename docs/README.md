 ## 项目特色

- **私钥不出服务器** — 密钥在本地生成，平台只下发签名证书
- **一键部署** — 支持 Nginx / Apache / IIS 自动识别与部署
- **自动续签** — 守护进程周期心跳，证书到期自动更新
- **安全通信** — HMAC-SHA256 签名 + nonce 防重放
- **代码开源** — Apache 2.0 许可证，可审计、可修改
