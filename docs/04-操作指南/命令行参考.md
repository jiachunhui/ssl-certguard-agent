 # 命令行参考

 > CertGuard Agent 所有命令行参数的完整说明

 ## 基本用法

 ```bash
certguard-agent [options]
```

 ## 参数列表

 | 参数 | 说明 | 示例 |
|------|------|------|
| `--token <t>` | 一次性注册令牌，仅首次运行需要 | `--token ct_reg_xxxxxx` |
| `--server <url>` | 平台 API 地址（默认 `http://localhost:5003`） | `--server https://api.topssl.cn` |
| `--update-secret <s>` | 更新 agent.json 中的 HMAC 密钥后退出 | `--update-secret new_secret_key` |
| `--update <url>` | 下载更新包并替换 Agent 二进制后退出 | `--update https://.../agent.zip` |
| `--data-dir <p>` | 数据目录路径 | `--data-dir /data/certguard` |
| `--heartbeat <s>` | 心跳间隔秒数（默认 60） | `--heartbeat 120` |
| `--register-only` | 仅注册后退出（安装脚本使用） | `--register-only` |
| `--insecure` | 跳过 SSL 证书验证（自签证书环境使用） | `--insecure` |
| `--uninstall` | 卸载 Agent | `--uninstall` |
| `--keep-data` | 配合 --uninstall 保留数据目录 | `--uninstall --keep-data` |
| `--version, -v` | 显示版本号 | `-v` |
| `--help, -h` | 显示帮助信息 | `-h` |

 ## 环境变量

| 变量 | 说明 |
|------|------|
| `CERTGUARD_TOKEN` | 注册令牌（替代 --token） |
| `CERTGUARD_SERVER` | 平台 API 地址（替代 --server） |
| `CERTGUARD_INSECURE=1` | 允许不安全 SSL 连接（替代 --insecure） |

 ## 典型使用场景

 ### 首次运行（注册）

 ```bash
certguard-agent --token ct_reg_xxxxxx
```

 如果只想注册然后退出（用于安装脚本）：

 ```bash
certguard-agent --token ct_reg_xxxxxx --register-only
```

 ### 更新 HMAC 密钥

 ```bash
certguard-agent --update-secret <新密钥>
```

 执行后需要重启服务使新密钥生效。

 ### 更新 Agent 版本

 ```bash
certguard-agent --update https://agent.topssl.cn/download/certguard-agent-linux-x64.zip
```

 Agent 会：

 1. 下载更新包并验证
 2. 备份当前版本到备份目录
 3. 解压新版本替换文件
 4. 自动退出，由更新脚本重启服务

 ### 查看版本

 ```bash
certguard-agent --version
# 输出: certguard-agent v1.1.3
```

 ### 自定义数据目录

 ```bash
certguard-agent --data-dir /data/certguard --token ct_reg_xxxxxx
```

 ### 自建平台环境

 ```bash
certguard-agent --server https://your-platform.com --insecure --token ct_reg_xxxxxx
```

 `--insecure` 用于平台使用自签证书时的开发测试环境。

 > 注意：除 `--register-only` 和 `--update` 等一次性操作外，日常运行时请不要附加任何参数，Agent 会自动加载 `agent.json` 中的配置。
