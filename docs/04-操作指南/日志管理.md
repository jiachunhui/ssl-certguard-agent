 # 日志管理

 > 如何查看、分析和管理 Agent 运行日志

 ## 日志文件位置

 Agent 日志存储在数据目录下的 `logs` 文件夹中：

 - **Linux**: `/var/lib/TopSSL-CertGuard-Agent/logs/`
 - **Windows**: `%ProgramData%\TopSSL-CertGuard-Agent\logs\`

 日志文件命名格式：

 ```
certguard-agent-20240101.log
certguard-agent-20240102.log
```

 ## 查看日志

 ### Linux

 ```bash
# 查看最新的 50 行日志
 tail -50 /var/lib/TopSSL-CertGuard-Agent/logs/certguard-agent-*.log

# 实时追踪日志
 tail -f /var/lib/TopSSL-CertGuard-Agent/logs/certguard-agent-*.log

# 通过 journalctl 查看
 journalctl -u topssl-certguard-agent -n 100 -f
 ```

 ### Windows

 ```powershell
# 查看最新的 50 行日志
 Get-Content "$env:ProgramData\TopSSL-CertGuard-Agent\logs\certguard-agent-*.log" -Tail 50

# 实时追踪日志
 Get-Content "$env:ProgramData\TopSSL-CertGuard-Agent\logs\certguard-agent-*.log" -Tail 10 -Wait
 ```

 ## 日志级别

| 级别 | 说明 | 示例场景 |
|------|------|----------|
| Information | 正常运行信息 | 心跳成功、任务完成、注册成功 |
| Warning | 需要关注的异常 | 平台连接失败（自动重试）、环境上报失败 |
| Error | 执行失败的操作 | 证书部署失败、配置检查失败 |
| Critical | 严重错误，Agent 退出 | 无身份信息、严重网络错误 |

 ## 日志轮转策略

| 配置项 | 值 |
|--------|------|
| 滚动方式 | 按天滚动 |
| 保留天数 | 7 天 |
| 单文件大小上限 | 10 MB |
| 超限处理 | 自动滚动至新文件 |
| 日志格式 | Serilog 结构化格式 |

 ## 日志内容示例

 ```log
[2024-01-01 12:00:00 INF] CertGuard Agent v1.1.3 启动中...
[2024-01-01 12:00:01 INF] 已加载身份信息: agt_xxxxxxxxxxxx
[2024-01-01 12:00:01 INF] 就绪。Web=nginx, 系统=Linux, 心跳=60s
[2024-01-01 12:01:01 INF] 发送心跳（版本=1.1.3）
[2024-01-01 12:05:00 INF] 获取到 1 个任务
[2024-01-01 12:05:00 INF] 任务 #42 类型=deploy_cert
[2024-01-01 12:05:01 INF] Nginx 证书已写入: example.com -> /etc/nginx/ssl/example.com
[2024-01-01 12:05:01 INF] Nginx 重载完成
[2024-01-01 12:05:01 INF] 任务 #42: success
 ```

 ## 日志中的关键信息

 关注以下关键事件：

- **注册成功**：Agent 首次注册的记录
- **就绪信息**：Web 服务类型和系统类型
- **任务执行**：任务 ID、类型、结果
- **心跳失败**：网络连接问题的重要信号
- **版本不匹配**：Agent 需要自动更新

 ## 故障排查日志定位

 | 问题 | 日志中搜索关键词 |
|------|-----------------|
| Agent 无法启动 | `严重错误`、`Critical` |
| 注册失败 | `注册失败`、`RegisterAsync` |
| 心跳失败 | `平台无法连接`、`HttpRequestException` |
| 证书部署失败 | `deploy_cert` + `failed` |
| Web 服务重载失败 | `重载失败`、`graceful` |
| 自动更新问题 | `更新`、`update`、`版本不匹配` |
