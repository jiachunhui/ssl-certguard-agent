 # 构建发布

 > 从源码编译和打包 CertGuard Agent

 ## 前置条件

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) 或更高版本
- Git（拉取源码）

 ## 克隆代码

 ```bash
 git clone https://github.com/jiachunhui/ssl-certguard-agent.git
 cd certguard-agent
 ```

 ## 构建所有平台

 ```bash
 cd src/CertGuard.Agent

# Linux x86_64
 dotnet publish -c Release -r linux-x64 --self-contained -o ../../build/linux-x64

# Linux ARM64
 dotnet publish -c Release -r linux-arm64 --self-contained -o ../../build/linux-arm64

# Windows x86_64
 dotnet publish -c Release -r win-x64 --self-contained -o ../../build/win-x64
 ```

 ### 指定版本号

 ```bash
# 构建时指定版本
 dotnet publish -c Release -r linux-x64 --self-contained `
   -p:Version=1.1.3 -p:AssemblyVersion=1.1.3.0 `
   -o ../../build/linux-x64
 ```

 ## 构建单个平台

 ```bash
# 构建当前平台
 dotnet publish -c Release --self-contained -o ../build/$(dotnet --info | grep RID | head -1 | awk '{print $NF}')

# 或指定平台
 dotnet publish -c Release -r linux-x64 --self-contained -o ../build/linux-x64
 ```

 ## 输出文件

 构建完成后，`build/` 目录下会生成：

 ```
 build/
 ├── linux-x64/
 │   └── certguard-agent          # Linux x86_64 可执行文件
 ├── linux-arm64/
 │   └── certguard-agent          # Linux ARM64 可执行文件
 └── win-x64/
     ├── certguard-agent.exe      # Windows x86_64 可执行文件
     └── certguard-agent.pdb      # 调试符号文件
 ```

 使用 `--self-contained` 参数，输出为单文件，包含所有依赖。

 ## 构建配置说明

 `CertGuard.Agent.csproj` 中的关键配置：

 ```xml
 <PropertyGroup>
   <OutputType>Exe</OutputType>
   <TargetFramework>net8.0</TargetFramework>
   <RootNamespace>CertGuard.Agent</RootNamespace>
   <AssemblyName>certguard-agent</AssemblyName>
   <Version>1.1.3</Version>

   <!-- 单文件发布 -->
   <PublishSingleFile>true</PublishSingleFile>
   <SelfContained>true</SelfContained>
   <IncludeNativeLibrariesForSelfExtract>true</IncludeNativeLibrariesForSelfExtract>

   <!-- 调试符号嵌入 -->
   <DebugType>embedded</DebugType>
 </PropertyGroup>
 ```

 > **注意**：Self-contained 发布包体积较大（约 40 MB），但无需在目标服务器上安装 .NET 运行时。

 ## 本地开发调试

 ### 使用 Visual Studio / Rider

 直接打开 `src/CertGuard.Agent/CertGuard.Agent.csproj`，按 F5 启动调试。

 ### 命令行调试

 ```bash
 cd src/CertGuard.Agent
 dotnet run
 ```

 ### 指定调试参数

 ```bash
 dotnet run -- --server http://localhost:5003 --token ct_reg_test --insecure
 ```

 `--insecure` 用于本地开发时跳过自签 SSL 证书验证。

 ## 发布到 NuGet

 本项目为自包含可执行文件，不是 NuGet 包，无需发布到 NuGet。构建产出直接作为二进制发布。

 ## CI/CD 建议

 建议使用 GitHub Actions 自动构建和发布：

 1. 推送 Tag 时触发构建
 2. 构建所有平台二进制
 3. 分别压缩为 zip 包
 4. 发布到 GitHub Releases

 > 详情请参阅 **[贡献指南](./贡献指南.md)** 了解提交流程。
