 # 系统要求

 > 在安装前确认您的服务器满足以下要求

 ## 操作系统

| 平台 | 架构 | 支持情况 |
|------|------|----------|
| Linux | x86_64 | ✅ 完整支持 |
| Linux | arm64 | ✅ 完整支持（如树莓派、AWS Graviton） |
| Windows | x86_64 | ✅ 完整支持 |
| macOS | - | ❌ 暂不支持（后续考虑） |

 ## 软件要求

 ### Linux

- 已安装 **Nginx** 或 **Apache**（至少一个）
- systemd 作为服务管理器（大部分现代发行版默认）
- `curl` 或 `wget`（用于下载安装包）

 ### Windows

- Windows Server 2016 / 2019 / 2022，或 Windows 10 / 11
- 已安装 **IIS**（Internet Information Services）
- PowerShell 5.1 或更高版本

 ## 网络要求

- Agent 需要能够访问 TOPSSL.CN 平台 API
- 默认 API 地址：`http://localhost:5003`（自建平台）
- 生产环境请配置正确的 API 地址
- 出站连接即可，无需开放入站端口

 ## 硬件要求

| 资源 | 最低要求 | 推荐 |
|------|----------|------|
| CPU | 1 核 | 1 核 |
| 内存 | 64 MB | 128 MB |
| 磁盘 | 50 MB | 100 MB |

 Agent 是一个轻量级守护进程，空闲时内存占用约 20 MB。

 ## 权限要求

| 操作 | 所需权限 |
|------|----------|
| 安装程序 | Linux：root / Windows：管理员 |
| 读写证书文件 | 对应 Web 服务的运行用户 |
| 重载 Web 服务 | root / 管理员 |
| 写入日志 | 对数据目录有写权限 |
