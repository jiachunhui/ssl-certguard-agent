 # 服务管理

 > 如何管理 CertGuard Agent 系统服务

 ## Linux (Systemd)

 ### 查看服务状态

 ```bash
 systemctl status topssl-certguard-agent
 ```

 ### 启动 / 停止 / 重启

 ```bash
 systemctl start topssl-certguard-agent
 systemctl stop topssl-certguard-agent
 systemctl restart topssl-certguard-agent
 ```

 ### 设置开机自启

 ```bash
 systemctl enable topssl-certguard-agent
 systemctl disable topssl-certguard-agent  # 取消开机自启
 ```

 ### 查看日志

 ```bash
# 查看最新日志
 journalctl -u topssl-certguard-agent -n 100

# 实时追踪日志
 journalctl -u topssl-certguard-agent -f

# 查看特定日期的日志
 journalctl -u topssl-certguard-agent --since "2024-01-01" --until "2024-01-02"
 ```

 ## Windows

 ### 查看服务状态

 ```powershell
 Get-Service TopSSLCertGuardAgent
 sc.exe query TopSSLCertGuardAgent
 ```

 ### 启动 / 停止 / 重启

 ```powershell
 Start-Service TopSSLCertGuardAgent
 Stop-Service TopSSLCertGuardAgent
 Restart-Service TopSSLCertGuardAgent

# 或使用 sc.exe
 sc.exe start TopSSLCertGuardAgent
 sc.exe stop TopSSLCertGuardAgent
 ```

 ### 设置开机自启

 ```powershell
 sc.exe config TopSSLCertGuardAgent start=auto
 sc.exe config TopSSLCertGuardAgent start=disabled  # 禁用开机自启
 ```

 ### 查看日志

 ```powershell
 Get-Content "$env:ProgramData\TopSSL-CertGuard-Agent\logs\certguard-agent-*.log" -Tail 100
 ```

 ## 常见服务问题

 | 问题 | 排查方法 |
|------|----------|
| 服务启动失败 | 查看日志：`journalctl -u topssl-certguard-agent -n 50` 或检查 Windows 事件查看器 |
| 服务频繁重启 | 使用 `--server` 参数检查平台地址是否正确，网络是否可达 |
| 服务启动后退出 | 确认注册信息完整，检查 `agent.json` 内容 |
| Windows 服务无法安装 | 确认以管理员权限运行 PowerShell |

 > 遇到服务问题？请查阅 **[常见问题](../06-常见问题/README.md)**。
