 ## 选择合适的教程

 Agent 在首次启动时会自动检测 Web 服务类型：

- 检测顺序：**Nginx → Apache → IIS**
- 如果只安装了其中一种，Agent 会自动选择对应的部署提供程序
- 如果多种服务共存，以检测到的第一个为准

 ## 生产环境建议

 - **先测试后上线**：在测试环境验证后再部署到生产服务器
 - **配置正确的心跳间隔**：默认 60 秒，可根据需要调整（建议 30-300 秒）
 - **监控服务状态**：配置外部监控检查 Agent 服务是否正常运行
 - **定期查看日志**：建议每天检查一次日志，确认证书部署正常
